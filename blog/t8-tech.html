What is Rails?

Rails is an open source web application framework for Web deveolopment in Ruby. Being open source, it promotes a universal access via a free license to a product's design or blueprint, and universal redistribution of that design or blueprint, including subsequent improvements to it by anyone. This structure enables a self-enhancing diversity of production models, communication paths, and interactive communities. It fosters a collaborative effort where programmers improve upon the source code and share the changes within the community so that other members can help improve it further.

The Rails philosophy is "Convention over Configuration." This means that a developer only needs to specify unconventional aspects of the application. This approach may seem simple, but its ramifications are enormous. No longer are developers working hard to reinvent the wheel. Their efforts are better spent here, improving, expounding upon, and streamlining already existing code. The result, aside from accelerated innovation, is less code and less repitition.

Changes made to applications are immediately applied, avoiding the time consuming steps normally associated with the web development cycle. The Rails framework has been measured to be 5-10 faster than comparable Java-based frameworks.

Many high-profile consumer web firms are using Ruby on Rails to build scalable, agile web applications. Some of the largest sites running Ruby on Rails include Twitter, GitHub, Scribd, Hulu, Yammer, Basecamp, and Shopify.
