How do I ask a good question?

Below is a question that was posed by one of my cohorts, James Fallon:

When should I use function expressions and when should I use function declarations? Is it just to use the variable somethingElse later on?

Function declaration:

function something() {

alert("Yo Yo!");

}

Function expression:

var somethingElse = function () {

alert("Yo Yo Ma");

};ï»¿

This is a question that has come up quite often since we started our unit on JavaScript.

One of the best things about this question is that it provides sample code, so there isn't any confusion over what he's inquiring about.

It's very concise; not overly verbose.

The question would have benifited from a title or headline, in order to grab attention. The context of the question was clear. A problem statement would have been beneficial. The spelling, grammar and formatting were in good shape. I think James made a good impression with the inquiry. I'd say this is a good example of how to interface with fellow cohorts in finding answers to questions and creating a team effort.
